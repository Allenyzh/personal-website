InputDateMask is a custom Vue 3 component built on top of PrimeVue, combining the power of masked input and inline date picking. It allows users to input or select dates in a consistent, validated, and formatted manner. This component is part of the Prime Kit open-source UI collection and can be installed via pnpm or npm

## Installation

<CodeBlockWithCopy code={`pnpm add primevuekit`} lang="bash" />
